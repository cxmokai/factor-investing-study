# 因子投资学习计划

> **目标市场：** 传统金融（A股）+ 加密货币
>
> **目标：** 从零开始掌握因子投资，最终能实盘交易
>
> **总时长：** 69 天，每天 2 小时

---

## 四个学习轨道

```
金融基础（11天）→ 回答"为什么"
  股票是什么、市场怎么运作、CAPM/FF3 的含义、行为金融

数学基础（10天）→ 回答"用什么工具"
  回归、假设检验、协方差矩阵、优化

因子投资（18天）→ 回答"怎么做"
  如何定义因子、如何检验因子、如何组合因子、如何构建组合
  如何用 ML 做因子合成、如何监控因子衰减

编程实战（30天）→ 回答"代码怎么写"
  从数据获取到因子研究到 ML 合成到模拟实盘，全流程代码实现
```

学习顺序：**金融基础 → 数学基础 → 因子投资 → 编程实战**

---

## 各轨道概览

### 1. 金融基础（11 天）

| 模块           |   天数   | 核心内容                                 |
| -------------- | :------: | ---------------------------------------- |
| 证券与市场基础 | Day 1-2  | 股票本质、A 股/加密交易规则、交易成本    |
| 财务报表与估值 | Day 3-4  | 三张表、PE/PB/ROE/EPS                    |
| 收益与风险度量 |  Day 5   | 收益率、波动率、VaR、夏普比率            |
| 投资组合理论   | Day 6-7  | 分散化、有效前沿、Markowitz              |
| 资产定价模型   | Day 8-10 | CAPM、APT、FF3、因子投资起源             |
| 行为金融       |  Day 11  | 过度反应、反应不足、因子存在的行为学解释 |

> 金融基础编号 Day 1-11 与全局编号一致。

**主要资料：** 《投资学精要》第11版、《漫步华尔街》第13版、《手把手教你读财报》、《因子投资：方法与实践》Ch1

### 2. 数学基础（10 天）

| 模块       |   天数    | 核心内容                          |

| 模块               |   天数    | 核心内容                                             |

**主要资料：** 《因子投资：方法与实践》Ch2-7、经典论文（FF1993、Jegadeesh & Titman 1993 等）
